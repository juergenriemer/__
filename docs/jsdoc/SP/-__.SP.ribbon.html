addTempGroup(args) → {Node}

Temporarily adds ribbon group to an existing ribbon on-the-fly.
It adds an ID to the LI element starting with "Temp" plus the groups label joined by dots (e.g. Temp.Task.Actions)

addTempIcon(args) → {Node}

Temporarily adds a ribbon icon into an existing ribbon group on-the-fly.
It will try to find an icon image path by looking upt the image map (__.SP.icon.mp) using the lowercase label name (replacing spaces with underscores) and defaults to the placeholder icon (orange dot)
Parameters:
Name Type Description
args Object a parameter object holding the following values
Properties
Name Type Attributes Description

dnGroup Node DOM node of the group the icon should get appended. (Note: you get the node by ID of the LI element)
fnAction function Function to be executed on click

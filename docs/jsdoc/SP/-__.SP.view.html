(async) add(args)

Adds a new view to a list
Parameters:
Name Type Description
args Object a parameter object holding the following values
Properties
Name Type Attributes Description
sList String name or guid of a list
sView String name of the view
lsFields Array array of field names to be displayed in the view
sTotal String <optional>
 field name for which totals should be displayed (NB: ATM only one field is supported)
jsLink String <optional>
 URL of a JSLink file to be loaded with this view.
bPublic Boolean <optional>
 flag whether the view is public or personal (default is false/personal)
bDefaultView Boolean <optional>
 flag whether the view is set to be default (default is false)
nRows Number <optional>
 number of items to be shown on one page (default is 10)
bPaging Boolean <optional>
 flag whether the view shows pagination (default is false)

(static) display(args)

Updates a list field with the following information
- display name
- flag to be required
- flag to be hidden
- flags in which forms to be shown
Parameters:
Name Type Description
args Object a parameter object holding the following values
Properties
Name Type Description
sList String name or guid of the list
sField String internal name of the field
sTitle String new display name of the field
bRequired Boolean flag whether a field is required
bHidden Boolean flag whether to hide a field
bNew Boolean flag whether to not load in new form
bEdit Boolean flag whether to not load in edit form
bDisp Boolean flag whether to not load in view form

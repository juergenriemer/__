 name of main action button (OK/Save), indicatng "null" will not render the button (default is "Save")

 name of cancel button, indicating "null" will not render the button (default is "Cancel" )

 flag wether a button to close the modal window is rendered (X in top right corner)

 message to be shown when {args.fnact} is called or when {args.bShowProcessing} is flagged true (default is "Processing your request.")

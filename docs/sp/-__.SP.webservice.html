(static) call(args) → {Object}

Calls a webservice and handles results

args Object a parameter object holding the following values

Name Type Attributes Description
sService String name of the webservice which is looked up in the O$C3.Url mapping object.
sEndpoint String name of the endpoint
sMethod String <optional>
 HTTP method (default is "POST")
oPayload Object <optional>
 Object holding parameters as key value pairs
aHeaders Object <optional>
 Object holding custom HTTP headers as key value pairs
msTimeout Number <optional>
 Timeout in millisecons (default is 20 seconds)

 Flag to indicate we are not interested in error handling.

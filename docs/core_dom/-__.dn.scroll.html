(static) on(dn_or_cb, cb_or_ms, msopt) → {Object}

Registers to the scroll event of either a DOM node or the entire page.
A callback function is invoked with a data object providing information about the current position which is, if not manually set, called every 300 milliseconds.
Parameters:
Name Type Attributes Description

cb_or_ms String | Number Either a callback function or a custom notification delay in milliseconds

 A notification delay in milliseconds
